#### Description (00)

Cooperative Education Workplace detail.

#### Requirements (00)

1. Name of workplace
2. Address
3. Contact details (e.g. name, position, address)
4. Employer Supervisor details (e.g. name, position, address)

#### When to send (00)

Before have a intern. Return to Cooperative Education, Education Service, Room 105 Building 1. Kasetsart University Bangkhen campus
